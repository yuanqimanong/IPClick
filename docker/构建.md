# 构建

> docker buildx build -f docker/Dockerfile -t ipclick:latest .

# 运行

> docker run -d -p 5555:9527 --name ipclick ipclick:latest

> ### config.toml
> docker run -d -p 5555:9527 --name ipclick -v /【你的路径】/:/root/.ipclick/ ipclick:latest


---

# image

## 导出

> docker save -o ipclick.tar ipclick:latest

## 导入

> docker load -i ipclick.tar
